## How It Works

1. Click the "Generate" button
2. Get a random 12-character password containing:
   - Uppercase letters (A-Z)
   - Numbers (0-9)
   - Symbols (!@#$%^&\*()\_+-=)

## Technical Details

- Simple random character selection algorithm
- Basic DOM manipulation to display results
